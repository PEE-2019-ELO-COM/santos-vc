# **Display de 7 segmentos**  
O código tem como finalidade fornecer a sequência de de leds a ser acendida num display de 7 segmentos catodo comum.  
Para isso é feita a entrada de um número  pelo usuário, que é posteriormente operado na forma binária, bit a bit utilizando a lógica feita pelos mapas de Karnaugh.
